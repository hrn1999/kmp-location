# kmp-location

Explorando como utilizar geolocalização no Kotlin Multiplatform.

## Plataformas

- Android
- iOS

## Stack

- Kotlin Multiplatform
- Compose Multiplatform
- Geolocation APIs nativas

## Build

```shell
# Android
./gradlew :composeApp:assembleDebug

# iOS
# Abra iosApp/ no Xcode
```
